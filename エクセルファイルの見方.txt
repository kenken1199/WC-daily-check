WC分析ツール　エクセルファイルの見方
========================================

このエクセルファイルは、重量チェッカーで測定したデータを自動で集計・グラフ化したものです。
ロットごとに1つのファイルが作成されます。

まず最初に見るシート：「分析レポート」
----------------------------------------
ファイルを開くと一番左に「分析レポート」というシートがあります。
統計数値・ランクコード集計・グラフがすべてここにまとまっているので、
基本的にはこのシートだけ見れば大丈夫です。


【基本統計】の見方
  ・全数          … 測定した個数（全部の件数）
  ・OK数          … 正量（合格）と判定された個数
  ・NG数          … 軽量または過量だった個数
  ・不良率(%)      … NG数 ÷ 全数 ×100。この数字が大きいほど不良が多い

  製造情報
  ・開始時刻／終了時刻／製造時間 … そのロットを作っていた時間帯

  統計値（OK品のみ）
  ・平均(g)        … OK品だけの重さの平均値
  ・標準偏差(g)    … 重さのばらつきの大きさ。小さいほど安定して同じ重さに仕上がっている
  ・Max(g)／Min(g) … OK品の中で一番重かった／軽かった値
  ・下限−3σ／上限+3σ … 「平均からどれくらいまでのばらつきなら通常範囲か」を示す目安の線。
                        統計的には、ここからはみ出るデータは1000個に3個ほどしか出ないはずの
                        「珍しい値」という意味になる
  ・外れ値件数     … 上の±3σの範囲からはみ出したOK品の個数
                    （多い場合は工程に何か変化があった可能性）


【ランクコード集計】の見方
  ・OK         … 正量（合格）
  ・軽量       … 規定より軽かったもの
  ・過量       … 規定より重かったもの
  ・２個乗り   … 2個が重なって計量されたと判定されたもの


【グラフ】の見方（分析レポートシートに埋め込まれています）

  ・全データ時系列 … OK・軽量・過量をすべて時間順に表示したグラフ。
      青丸      ：OK品
      オレンジ下三角（▽）：軽量
      赤上三角（△）：過量
      赤い実線  ：OK品の平均値
      オレンジ破線：±3σの範囲（OK品のみで計算）
      不良がどの時間帯に出ているかを見るのに使う

  ・時系列チャート（OK品のみ） … OK品だけを時間順に並べたグラフ。
      青丸      ：OK品
      赤い×印  ：外れ値（OK判定だが±3σを超えた重量の品）
                  最も重い外れ値・最も軽い外れ値には重量の数値が表示される
      赤い実線  ：OK品の平均値
      オレンジ破線：±3σの範囲（このはみ出しが多いと工程が安定していない）


ほかのシートについて
----------------------------------------
  ・全データ時系列           … 上記グラフの拡大表示用シート
  ・全データ                 … 測定した全件の生データ
                               （測定値出力No.・日付時刻・測定値・ランクコードの4列）
                               軽量・過量の行は赤色で表示
  ・ヒストグラム（OK品のみ） … OK品の重さの分布を山型グラフで表示したもの（拡大表示用）
  ・時系列チャート（OK品のみ）… 上記グラフの拡大表示用シート
  ・全OKデータ               … OK品だけの生データ一覧（外れ値は赤色で表示）
  ・全外れ値（±3σ超）       … ±3σの範囲を超えた測定値の一覧


困ったときは
----------------------------------------
・不良率が高い、外れ値が多い場合 → いつ頃から増えたか「全データ時系列」グラフで確認
・「分析レポート」の数字の意味が分からない場合 → このファイルを見返してください
