WC分析ツール　エクセルファイルの見方
========================================

このエクセルファイルは、重量チェッカーで測定したデータを自動で集計・グラフ化したものです。
ロットごとに1つのファイルが作成されます。

まず最初に見るシート：「分析レポート」
----------------------------------------
ファイルを開くと一番左に「分析レポート」というシートがあります。
ここに重要な情報がすべてまとまっているので、基本的にはこのシートだけ見れば大丈夫です。

【基本統計】の見方
  ・全数          … 測定した個数（全部の件数）
  ・OK数          … 正量（合格）と判定された個数
  ・NG数          … 軽量または過量だった個数
  ・不良率(%)      … NG数 ÷ 全数 ×100。この数字が大きいほど不良が多い

  製造情報
  ・開始時刻／終了時刻／製造時間 … そのロットを作っていた時間帯

  統計値（OK品のみ）
  ・平均(g)        … OK品だけの重さの平均値
  ・基準値(g)      … 規格値を入力した場合のみ表示。狙いとする重さ
  ・基準値とのズレ(g) … 平均が基準値よりどれだけ重い/軽いか
  ・標準偏差(g)    … 重さのばらつきの大きさ。小さいほど安定して同じ重さに仕上がっている
  ・Max(g)／Min(g) … OK品の中で一番重かった／軽かった値
  ・下限−3σ／上限+3σ … 「平均からどれくらいまでのばらつきなら通常範囲か」を示す目安の線。
                        統計的には、ここからはみ出るデータは1000個に3個ほどしか出ないはずの
                        「珍しい値」という意味になる
  ・外れ値件数     … 上の±3σの範囲からはみ出した個数（多い場合は工程に何か変化があった可能性）

  規格値（規格値を入力したロットのみ表示）
  ・UCL／LCL       … 製品としてOKとする上限・下限の重さ（社内規格や取引先との約束値）
  ・UCL超え件数／LCL未満件数 … 規格を外れていた個数（全データのうち）

【ランクコード集計】の見方
  ・OK         … 正量（合格）
  ・軽量       … 規定より軽かったもの
  ・過量       … 規定より重かったもの
  ・２個乗り   … 2個が重なって計量されたと判定されたもの

【グラフ】の見方
  ・時系列チャート（OK品のみ） … OK品だけを時間順に並べたグラフ。
      赤い線：OK品の平均値
      オレンジの点線：±3σの範囲（このはみ出しが多いと工程が安定していない）
      紫の一点鎖線：UCL／LCL（規格値を入力した場合のみ）
  ・全データ時系列 … OK・軽量・過量をすべて時間順に表示したグラフ。
      青丸：OK　／　オレンジ三角：軽量　／　赤三角：過量
      不良がどの時間帯に集中して出ているかを見るのに使う

ほかのシートについて
----------------------------------------
  ・全データ時系列 … 上記グラフの拡大表示用シート
  ・全データ      … 測定した全件の生データ（軽量・過量は赤色で表示）
  ・ヒストグラム（OK品のみ） … OK品の重さの分布を山型グラフで表示したもの
  ・時系列チャート（OK品のみ） … 上記グラフの拡大表示用シート
  ・全OKデータ    … OK品だけの生データ一覧（外れ値は赤色で表示）
  ・全外れ値（±3σ超） … ±3σの範囲を超えた測定値の一覧

困ったときは
----------------------------------------
・不良率が高い、外れ値が多い場合 → いつ頃から増えたか「全データ時系列」グラフで確認
・「分析レポート」の数字の意味が分からない場合 → このファイルを見返してください
